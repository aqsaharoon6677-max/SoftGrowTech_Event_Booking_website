# Event Booking Website - SoftGrowTech

## Internship Task 1 - SoftGrowTech

###  Description
A responsive event booking website where users can browse and book events across Pakistan. Built as part of my Web Development Internship at SoftGrowTech.

###  Technologies Used
- HTML5 - For structure
- CSS3 - For styling and responsive design  
- JavaScript - For interactivity and form validation
- Font Awesome - For icons

###  Features
- Browse events by categories
- Detailed event pages
- Booking form with validation
- Thank you page after booking
- Fully responsive on mobile, tablet & desktop
- Clean and modern UI/UX
